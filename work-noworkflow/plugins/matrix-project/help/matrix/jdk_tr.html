Yapılandırmaların çalıştırılacağı JDK(ları) belirleyiniz. Eğer, herhangi biri seçilmez ise varsayılan JDK kullanılacaktır (JAVA_HOME'un olmadığı, ve java komutunun PATH içerisinde olduğu varsayılmaktadır.). Eğer birden fazla JDK seçilirse, konfigürasyon matrisi belirtilen tüm JDK'ları içerecektir.
Eğer yapılmak istenen testin, farklı JDK'lar üzerinde çalıştırılması isteniyorsa, birden fazla değer seçmek kullanışlı olacaktır,
Yapılandırma esnasında, seçilen JDK değeri "jdK" ekseni olarak görülecektir. Eksen değerlerine nasıl erişileceğine dair daha fazla bilgi için, aşağıda "eksenler" ile ilgili yardım kısmına bakınız.
